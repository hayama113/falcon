# ぶんぐのたまご / PWAプロトタイプ

文房具の妖精をたまごから育てる、幼児〜中学生の女の子向け育成アプリのプロトタイプです。

## 入っている機能

- たまごから「ぴつる」へ成長
- タップ育成
- ごはん、そうじ、おべんきょう、デコる、ねる
- ステータス：きれいさ、やる気、なかよし、ひらめき、おしゃれ
- レベルアップ、5段階進化
- けしごむそうじミニゲーム
- 図鑑
- 日記
- 名前変更
- localStorage保存
- PWA対応：manifest.json / service worker / アイコン

## 使い方

1. `index.html` をブラウザで開く
2. GitHub Pagesなどにアップロードすると、iPhone/Androidのホーム画面に追加できます
3. iPhoneの場合はSafariで開き、「共有」→「ホーム画面に追加」

## 注意

- 既存キャラクターや既存育成玩具の名称・デザインは使っていません
- 外部通信、広告、アカウント登録は未実装です
- 子ども向け本番アプリにする場合は、課金・広告・個人情報・保護者同意の設計を別途確認してください
